Space_Shooter
=============

Simple shoot'em up space shooter


Requires: 
-Visual studio 2010
-XNA 4.0
-Microsoft Kinect SDK 1.6

To run:
1. download or clone repository
2. run solution file, eg. "2DShootertutorial.sln"
3. enjoy
